# Cursor AI를 이용한 금속공예 공방 홈페이지 만들기

해당 프로젝트는 Cursor AI를 통해 제작된 프로젝트입니다.

## 사전작업
```bash
# npm
npm install
```

## 서비스 실행하기
로컬 URL : `http://localhost:3000`
```bash
npm run dev
```
